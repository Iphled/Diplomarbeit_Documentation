GraphClassification.PetriLyzer.ExplainClassification()¶

Opens a new window explaining the classification result.

Parameters¶

None

Returns¶

None

Notes¶

This function opens a new window with the following contents: - A label with the text ‘Your Petri Net:’ - A display of the Petri net - A label with the text ‘Classification: ‘ followed by the classification result - A sequence of toggle frames, each containing an explanation of a single step of the classification process.
