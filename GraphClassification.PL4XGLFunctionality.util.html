GraphClassification.PL4XGLFunctionality.util.eval_val.find_default_and_fitted_label(data, dataset)¶

Find the default label and fitted label for the given dataset.

The default label is the label with the most graphs that are not described by any learned GDL program. The fitted label is the label with the most incorrect classifications.

Parameters¶

dataData

The data object

datasetstr

The name of the dataset

Returns¶

tuple

A tuple containing the default label, the fitted label, and the number of graphs that are not described by any learned GDL program.

GraphClassification.PL4XGLFunctionality.util.eval_val.search_hyperparameters(data, dataset)¶

Search for the best hyperparameters for the given dataset.

The best hyperparameters are chosen by first finding the default label and the fitted label, and then searching over possible amplification factors and validation thresholds.

Parameters¶

The data object
