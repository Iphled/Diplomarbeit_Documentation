GraphClassification.PetriLyzerFunctionality.util.classification.ClassificationClasses.getExplanationFrames(classification, result, root)¶

Converts a classification and a result into a list of ToggledFrame objects. These ToggledFrame objects contain the programs of the classification with their respective priorities and labels.

The ToggledFrame objects are colored in green if the program’s label matches the result and red otherwise.

Parameters:

  • classification (ClassificationGoal) – The classification

  • result (int) – The result to compare the programs with

  • root (tkinter.Tk) – The root window for the ToggledFrame objects

Returns:

A list of ToggledFrame objects

Return type:

list

GraphClassification.PetriLyzerFunctionality.util.classification.ClassificationClasses.tuplesToClassificationGoals(tuples)¶

Converts a list of tuples into a list of ClassificationGoal objects

The tuples should have the following structure: (name,path,answers,objective) where name is the name of the classification goal, path is the path to the classification program, answers is a list of possible answers to the classification and objective is the type of the classification.

Parameters:

tuples – A list of tuples

Returns:

A list of ClassificationGoal objects
