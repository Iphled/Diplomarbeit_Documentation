GraphClassification.PetriLyzerFunctionality.util.gui.Display.extractInfo(node)¶

Extracts information from a node in a GDL program.

Given a node in a GDL program, this function will extract the information from it and return a string and a color. The string will be empty if the node is not a place or transition, and it will contain the range of possible markings if the node is a place or transition. The color will be green if the node is a place or transition, orange if it is a transition, blue if it is a place, and white if it is not a place or transition.

Parameters¶

node: dict

Node in a GDL program

Returns¶

res: str

String containing the information from the node

color: str

Color of the node
