evaluate(programs, Pl, index, postcondition)¶

Evaluate the certification using the given programs and preconditions.

The evaluation is done by iterating through the programs and applying the extend method to each program in the precondition. If any of the programs in the precondition overlap with the ExtendedGDL object, the evaluation returns False. Otherwise, the evaluation returns the postcondition.

Parameters:

  • programs (list) – The list of GDL programs.

  • Pl (list) – The list of indices of the GDL programs that are in the precondition.

  • index (int) – The index of the current GDL program.

  • postcondition (list) – The postcondition of the certification.

Returns:

The result of the evaluation.

Return type:

bool

extend(programs, Pl, index, postcondition, mapping: list = [])¶

Extend the certification by applying the given mapping to the current GDL program.

The method first computes the mapping between the node variables of the ExtendedGDL object and the node variables of the GDL program. It then iterates over the mapped node variables and the edge variables of the GDL program and depletes the intervals of the ExtendedGDL object with the intervals of the GDL program. The method then calls the extendOrEvaluate method with the depleted ExtendedGDL object and the remaining mapping.

The method returns True if the certification is successful, False otherwise.

Parameters:

  • programs (list) – The list of GDL programs.

  • Pl (list) – The list of indices of the GDL programs that are in the precondition.

  • index (int) – The index of the current GDL program.

  • postcondition (list) – The postcondition of the certification.

  • mapping (list) – The mapping between the node variables of the ExtendedGDL object and the node variables of the GDL program.

Returns:

The result of the extension.

Return type:

bool

extendOrEvaluate(programs, Pl, index, postcondition, mapping)¶

Extend or evaluate the certification using the given mapping.

If the length of the mapping is less than 2, the method evaluates the certification by calling the evaluate method. Otherwise, it extends the certification by calling the extend method.

Parameters:

  • programs (list) – The list of GDL programs.

  • Pl (list) – The list of indices of the GDL programs that are in the precondition.

  • index (int) – The index of the current GDL program.

  • postcondition (list) – The postcondition of the certification.

  • mapping (list) – The mapping between the node variables of the ExtendedGDL object and the node variables of the GDL program.

Returns:

The result of the evaluation or extension.

Return type:

bool

Check if the given mapping is valid with respect to the given program.

The mapping is valid if for every node variable in the program, the intervals in the ExtendedGDL object are a subset of the intervals in the program, and for every edge variable in the program, the intervals in the ExtendedGDL object are a subset of the intervals in the program.

GraphClassification.PetriLyzerFunctionality.util.certification.ParseCertificate.evaluateEdge(tokens, var, num)¶

Evaluates an edge of the GDL program given in lexical representation.

The function transforms an edge of the GDL program given in lexical representation into a tuple (interval, node1, node2) where interval is a dictionary which maps the interval names to the intervals, node1 is the name of the first node of the edge and node2 is the name of the second node of the edge. It returns a tuple (interval, node1, node2).

Parameters:

  • tokens (list) – The list of tokens

  • var (list) – The list of variable names

  • num (list) – The list of numbers

Returns:

A tuple (interval, node1, node2) where interval is a dictionary which maps the interval names to the intervals, node1 is the name of the first node of the edge and node2 is the name of the second node of the edge

Return type:

tuple
